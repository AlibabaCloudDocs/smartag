# Renew and Change Specification

After a Smart Access Gateway \(SAG\) instance expires, you can renew it and upgrade or downgrade its specification. The new specification takes effect when the renewed subscription period starts. It does not take effect immediately.

1.  Log on to the [Smart Access Gateway console](https://smartag.console.aliyun.com).

2.  On the Smart Access Gateway page, find the target instance and click ![更多](https://static-aliyun-doc.oss-accelerate.aliyuncs.com/assets/img/en-US/4492404061/p101595.png) in the **Actions** column.

3.  Click **Renew and Change Specification**.

4.  On the Renew and Change Configuration page, set the **Peak Bandwidth** and **Subscription Duration**, select the **SAG Bandwidth Terms of Service** check box, and then settle the payment.
